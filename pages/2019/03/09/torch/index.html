PyTorch 之 torch 包

  • 1. 张量

torch 包里定义了多维数据结构及运算。也提供了很多高效序列化数据的函数。可在 GPU 上进行计算。

1. 张量

// 若 obj 是张量类型，返回 True
torch.is_tensor(obj)
// 是否为 storage 类型
torch.is_storage(obj)

// 设置浮点数默认类型，初始为 torch.float32
torch.set_default_dtype(d)
// 获得浮点数默认类型
torch.get_default_dtype() → torch.dtype

// 设置默认张量的类型，初始为 torch.FloatTensor
torch.set_default_tensor_type(t)

// 返回张量中元素数量
torch.numel(input) → int

// 设置打印格式
torch.set_printoptions(precision=None, threshold=None,
     edgeitems=None, linewidth=None, profile=None)
